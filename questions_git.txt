Activité: Utilisation de GitHub avec Visual Studio Code

Quelques questions pour découvrir l'outil GitHub :

1.	Qu'est ce que Git ? Quelle différence avec GitHub ?

	Git est un logiciel permettant aux développeurs de sauvegarder instantanément.
	La différence avec GitHub est une plateforme web permettant d’utiliser Git permettant la collaboration.

2.	Qu'est ce qu'un Repository ?

	Un Repository est un lieu de stockage de fichier et l'historique des modifications.

3.	Qu'est ce qu'un commit ? Un push ? Un pull ?

	Un commit = permet d’envoyer les fichiers modifiés au préalable et pouvoir les renvoyer sur Github.
	push = envoie les commits vers le depots distant
	pull = récuperer et fusionner les modifications du dépôts distant.

4.	A quoi sert le fichier README.md dans un repository GitHub ?

	Le fichier sert a documenter un projet sur GitHub.

5.	Comment récupérer un repository GitHub en ligne de commande sur un invite de commande ?

	On utilise la commande git clone https://github.com/utilisateur/nom-du-repo.git
